
# CHU EIA

A small squelette app for chu project

## Tech Stack

**Server:** Java, Spring Boot, Spring Data


## Run Locally

Clone the project

```bash
  git clone git@github.com:moncifem/chu.git
```

Go to the project directory

```bash
  cd chu
```

Install dependencies

```bash
  mvn install
```

Start the server

```bash
  mvn package
```
and run the .jar /target directory
